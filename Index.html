Stage 1: Make your Web Page - HTML and CSS

1-1: The Basics of the Web and HTML

A user requests a web page from a server via HTTP, the server then sends the page to the users browser.
HTML = Hypertext Markup Language

Inline vs. Block Elements
Block elements such as div and p form a invisible box around
their contents.
Inline elements such as b and span do not form a invisible box around their contents.

1-2: Creating a Structured Document with HTML

Structured Documents
Note: This Wikipedia article perfects sums up what a structured document is ;
"A structured document is an electronic document where some
method of embedded coding, such as mark-up, is used to give the
whole, and parts, of the document various structural meanings according
to a schema. A structured document whose mark-up doesn't break the schema
and is designed to conform to and which obeys the syntax rules of its mark-up
language is "well-formed". "

Repetition
you can avoid uneccessary repetition in the code by using things such as syles and
classes in CSS. This is done to save time, make the code easier to read and
also easier to maintain.

For example: when you make a change to a single class in CSS, which is repeated
throughout the code all the same elements defined as that class will change.
This method is far easier then having to make many of the same changes throughout
the code individually, and helps prefvent errors throughout the code.
